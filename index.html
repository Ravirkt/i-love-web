Wat heb ik vandaag gedaan?

Vandaag heb ik geleerd over het maken van een Breakdown schets. Dit heb ik toegepast aan mijn schets. verder heb ik verder gewerkt aan mijn visitekaartje. De styling heb ik aangepast en ik heb basic animaties toegevoegd.

1.Wat is het nut van een breakdown schets Dit maakt duidelijk welke elementen en properties van css, html en javascript je kunt gaan gebruiken om de objecten van je schets werkelijk te maken

2.Waarop moet je letten als je een breakdown schets maakt Een duidelijke onderscheidt maken tussen de elementen en properties van de verschillende talen. dit kan met verschillende kleuren.
